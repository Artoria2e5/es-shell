We'd like to announce release 0.82 of es.  The main difference
between this version and 0.8 is that the <> for obtaining the return
value (exit status) of an es command has been renamed <=.  <> now
means redirection to and from a file.

Other changes include:
	signal handling has been cleaned up.
	some portability issues have been resolved.
	all reported bugs were fixed (we think).
See the file CHANGES or the man page for details.

We hope that there will be no major user-visible changes between
now and version 1.0; expected areas of change, in addition to bug
fixes, are performance and documentation.

The source is available as
	ftp.white.toronto.edu:/pub/es/es-0.82.tar.Z

We plan on announcing the availability of this release to various
usenet newsgroups soon, so if there are any serious problems, please
let us know.

Thank you.

-- Paul and Byron
