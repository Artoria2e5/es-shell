Non-command specific redirection, as in

	exec >[2] err.out

doesn't seem to have any effect in es.

e.g.,

	; cat tst
	exec >[2] err.out
	ls wakabunga
	;
	; es tst
	ls: wakabunga: No such file or directory
	;
	; rc tst
	; cat err.out
	ls: wakabunga: No such file or directory
