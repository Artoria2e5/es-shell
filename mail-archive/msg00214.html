We'd like to announce release 0.8 of es.  This version is much changed since
0.79, but is much more stable (we think) and (finally!) contains a man page.
We recommend that everybody who's using es upgrade to this version.

The source is available as
	ftp.white.toronto.edu:/pub/es/es-0.8.tar.Z

We plan on announcing the availability of this release to various usenet
newsgroups soon, so if there are any serious problems, please let us know.

Thank you and bon appetit.

-- Paul and Byron
